# Bank Seter Application

This is a Flask banking project using Python, SQL Server, and pyodbc.

## Files
- `app.py` - main Flask application
- `SQLQuery1.sql` - database script
- `SQLQuery2.sql` - database script

## How to run
1. Install packages:
   `python -m pip install flask pyodbc reportlab werkzeug`
2. Create database:
   `BankDB`
3. Run:
   `python app.py`

## Features
- Register account
- Login
- Deposit money
- Withdraw money
- Download statement PDF
